説明

日本ゲーム大賞アマチュア部門出場のためにUnityで作成し、最終審査まで残った作品です

ページをめくるというアクションを生かして捲る途中で止めることでオブジェクトを滑らせて動かし、それを利用して足場 を作ったり、障害物を消しながら進んでいくパズルアクションゲームです。

プログラム

実装箇所

プレイヤーがマスの上に移動する処理

マスの上にオブジェクトがあった場合、オブジェクトを移動対象として移動する処理

移動対象のオブジェクトと現在いるオブジェクトの高さを比較し、移動できるか判断する処理

ページをめくった場合、プレイヤーを持ち上げる処理

落下時に足元のオブジェクトを移動対象として格納する処理

アニメーションの遷移処理

操作キャラクターの各オブジェクトへの接触判定
